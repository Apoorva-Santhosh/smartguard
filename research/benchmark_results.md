## Decision
**Primary model: sumitranjan/PromptShield**

PromptShield was the only model among 5 candidates to correctly 
classify all three attack categories (jailbreak, injection, toxic) 
while correctly passing benign prompts. P95 latency on my hardware 
(CPU only) was 62.8ms — well within real-time API range.

**Eliminated:**
- toxic-comment-model: missed all attacks (non-toxic on meth synthesis)
- toxic-bert: confidence 0.002 on harmful content, not suited for adversarial prompts
- KoalaAI: 314ms P95, exceeds latency budget
- deberta-v3-base-injection: 893ms P95, unusable for real-time. Reserved 
  as the "accuracy-only" answer — best injection detection but too slow.

**Hypothetical answers:**
- If latency only: keyword filter (~0ms)
- If accuracy only: deberta-v3-base-injection (perfect injection detection, 
  ignore latency cost)
